# mcp2221a
MCP2221A interface library using the usb repo for zig lang

#Example usage.
1. Clone repo
2. `cd mcp2221a`
3. `zig build`
4. `sudo zig build run`

#Current
- It's current state only has one command for the chip i.e get_status.
